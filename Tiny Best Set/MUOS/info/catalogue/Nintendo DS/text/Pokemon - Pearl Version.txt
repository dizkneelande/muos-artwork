This is the Pearl version of Pokémon. The differences between the Diamond and Pearl versions include how often and which types of Pokémon appear. This game can communicate with the Diamond version using Nintendo Wi-Fi Connection and DS Wireless Communications. This game can also be connected to the Ruby, Sapphire, FireRed, LeafGreen, and Emerald versions of Pokémon using the Dual Slot feature of Nintendo DS. This game CANNOT be connected to Game Boy, N64, or GameCube versions of Pokémon. It is also not compatible with any Pokémon Battle-e Cards.
